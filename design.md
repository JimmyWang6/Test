# 网页版五子棋游戏设计文档

## 1. 项目概述

### 1.1 项目背景
五子棋（Gomoku）是一种经典的两人对弈棋盘游戏，玩家轮流落子，先在横、竖或斜方向连成五子的一方获胜。本项目旨在开发一款运行于浏览器的网页版五子棋游戏，无需安装任何插件，开箱即用。

### 1.2 项目目标
- 提供流畅、美观的网页五子棋对局体验
- 支持本地双人对战（玩家 vs 玩家）
- 提供清晰的游戏状态提示（当前回合、胜负判断）
- 支持悔棋、重置对局等基础操作

---

## 2. 功能需求

### 2.1 核心功能

| 功能模块 | 描述 |
|---------|------|
| 棋盘渲染 | 绘制 15×15 的标准五子棋棋盘，包含横竖线及天元/星位标记 |
| 落子 | 玩家点击交叉点落子，黑棋先手 |
| 回合切换 | 每次落子后自动切换到对方回合 |
| 胜负判断 | 实时检测横/竖/左斜/右斜四个方向是否连成五子 |
| 游戏结束提示 | 胜利时高亮获胜棋子并弹出提示 |
| 重新开始 | 一键清空棋盘、重置所有状态 |
| 悔棋 | 撤销上一步落子 |

### 2.2 扩展功能（后续版本）
- AI 对战（Minimax + Alpha-Beta 剪枝）
- 计时器（每步限时）
- 对局记录回放
- 在线联机对战（WebSocket）

## 3. 技术方案

### 3.1 技术栈

| 层次 | 技术 |
|------|------|
| 界面结构 | HTML5 |
| 样式 | CSS3（Flexbox 布局、过渡动画） |
| 逻辑 | 原生 JavaScript（ES6+） |
| 渲染方式 | HTML Canvas 2D |

> 选择纯前端方案（无框架、无后端依赖），保证单文件可运行、零环境配置。

### 3.2 模块划分

```
index.html          单入口文件
├── <style>         样式层：页面布局、棋子样式、动画
├── <canvas>        渲染层：棋盘与棋子绘制
└── <script>        逻辑层
    ├── GameState   游戏状态管理（棋盘数组、当前玩家、历史记录）
    ├── Renderer    棋盘与棋子绘制
    ├── WinChecker  胜负判断算法
    └── EventHandler 用户交互（点击/按钮）
```

## 4. 数据模型

### 4.1 棋盘状态

```js
// 棋盘二维数组：0=空, 1=黑棋, 2=白棋
board[row][col] = 0 | 1 | 2

// 棋盘尺寸
const BOARD_SIZE = 15;   // 15×15 格
const CELL_SIZE  = 40;   // 每格像素（px）
const PADDING    = 40;   // 棋盘边距
```

### 4.2 游戏状态对象

```js
{
  board: number[][],      // 当前棋盘
  currentPlayer: 1 | 2,  // 1=黑棋, 2=白棋
  history: Array<{        // 悔棋历史
    row: number,
    col: number,
    player: number
  }>,
  gameOver: boolean,
  winner: number | null
}
```

## 5. 核心算法

### 5.1 坐标转换

```
canvas 像素坐标 → 棋盘交叉点坐标

col = Math.round((x - PADDING) / CELL_SIZE)
row = Math.round((y - PADDING) / CELL_SIZE)
```

### 5.2 胜负判断

遍历落子位置，沿四个方向（水平、垂直、主对角、副对角）计数连续同色棋子，若 ≥ 5 则判定获胜。

```
方向向量：
  水平：    [0, +1]
  垂直：    [+1, 0]
  主对角：  [+1, +1]
  副对角：  [+1, -1]

计数函数：
  count(row, col, dr, dc) = 从(row,col)沿(dr,dc)方向连续同色棋子数
  + count(row, col, -dr, -dc) - 1   （双向合计，减去重复计算的起点）
```

## 6. UI 设计规范

### 6.1 视觉风格
- 棋盘：传统木纹黄（`#DCB47C`），线条深棕色
- 黑棋：径向渐变黑色（带高光）
- 白棋：径向渐变白色（带高光）
- 获胜棋子：红色光圈高亮

### 6.2 页面布局

```
┌────────────────────────────────┐
│         网页版五子棋            │
├────────────────────────────────┤
│  状态栏：当前玩家 / 胜负提示     │
├────────────────────────────────┤
│                                │
│         15×15 棋盘             │
│                                │
├────────────────────────────────┤
│   [悔棋]        [重新开始]      │
└────────────────────────────────┘
```

### 6.3 交互反馈
- 鼠标悬停时显示落子预览（半透明棋子）
- 落子动画：缩放弹入效果
- 胜利时棋盘变暗，获胜棋子高亮闪烁
